# Date 日期

```javascript
var date = new Date;  //取得現在時間
date.getHours();  //取得小時
date.getMinutes();  //取得分鐘
date.getSeconds();  //取得秒數
date.getMilliseconds();
date.getDate();
date.getDay();
date.getMonth();
date.getFullYear();
date.getTime();
```

** 月份是從0開始算，0 = 1月...
